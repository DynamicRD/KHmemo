JDK: 	Java Development Kit,자바로 만든 언어를 컴퓨터가 실행시켜줄수 있도록
	실행,이진(바이너리)코드로 변형시켜주는것,번역기 같은것,=컴파일러

JRE:	Java Runtime Enviroment,JDK속에 들어있음.자바로 만든 코드를 PC로 
	실행할수 있도록 환경을 제공해주는 프로그램.

JVM:	Java Virtual Machine,자바 프로그램을 어떤 운영체제건 돌릴수 있게 
	만들어주는 프로그램,만능키

쿠키: 	서버에 저장해야할 정보를 내 컴퓨터에 대신 저장,웹사이트를 방문할때만

path:	실행하는 명령어를 아무 위치(폴더)에서 쓸 수있는 권한을 설정

cmd:	명령어를 쓸수있는창,command

cd: 	change directory,이동

dir: 	directory,하위 디렉토리 보기

"":	더블 쿼테션 마크,띄어쓰기 필요할때 이용

dll:	컴포넌트,동적 라이브러리,하인?

javac: 	java로 된 파일을 class로 컴파일
	class파일은 아무 운영체제로 바꿀수있음
