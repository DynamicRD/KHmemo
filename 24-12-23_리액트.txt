React
자바스크립트의 UI 라이브러리

클라이언트에게 데이터를 미리 에이잭스 방식으로 자바스크립트가 받아옴
그 데이터로 리액트가 화면을 만들고 웹브라우저에 전달함

서버에서 해야할일을 각각의 클라이언트가 담당하게 한다.
jsp대신 한다
자바코드가 없고 자바스크립트 코드로만 
웹서버에 데이터 요청해서 화면에 전달하는 역할까지 담당

React.js를 구동시키는 환경이 Node.js(톰캣서버)
원래 React는 웹브라우저에서만 돌아감

===================================
자바스크립트
6가지의 타입을 가진다
기본타입
	Number/String/Boolean/Undefined/Null
객체타입
	Object/Array/Function

자바스크립트는 오른쪽의 상수값에 의해 타입이 결정된다.
오른쪽의 값이 없으면 Undefined 타입(타입이 정해지지 않았어요)
자바스크립트는 디폴트 값이 없다
자바스크립트의 NULL은 나는 아무것도 값을 가진게 없어요 표시(번지를 가르키지 않는다)

객체타입(클래스) Object
배열 Array
함수 Function'

객체 선언방법

let num = { };
구성 멤버변수는 다 public
자바스크립트는 객체에 멤버변수를 추가할 수도 있다.
