
===================================
자바스크립트
6가지의 타입을 가진다
기본타입
	Number/String/Boolean/Undefined/Null
객체타입
	Object/Array/Function

자바스크립트는 오른쪽의 상수값에 의해 타입이 결정된다.
오른쪽의 값이 없으면 Undefined 타입(타입이 정해지지 않았어요)
자바스크립트는 디폴트 값이 없다
자바스크립트의 NULL은 나는 아무것도 값을 가진게 없어요 표시(번지를 가르키지 않는다)

객체타입(클래스) Object
배열 Array
함수 Function'

객체 선언방법

let num = { };
구성 멤버변수는 다 public
자바스크립트는 객체에 멤버변수를 추가할 수도 있다.

자바스크립트는 모든 객체에 true false를 준다

원시객체타입은 불변값이다
let p1 =1;
let p2 = p1; 
p2 = 2

여기에선 p2값을 지워서 바꾸지 못하고 이동해서 2를 준다

얕은 복사(주소를 공유한다)
